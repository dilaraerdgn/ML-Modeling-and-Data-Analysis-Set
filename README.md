# Veri Bilimi ve Makine Öğrenmesi Proje Seti (1-2-3-4)

Bu çalışma, ham bir veri setinin işlenmesinden başlayarak, en uygun makine öğrenmesi modelinin seçilmesine kadar geçen tüm "Data Pipeline" süreçlerini kapsayan kapsamlı bir analiz projesidir.

## Proje Aşamaları

1. **Veri Keşfi (EDA):** Veri setinin yapısı incelenmiş, temel istatistiksel analizler yapılmıştır.
2. **Veri Ön İşleme (Pre-processing):**
   - Aykırı değerlerin (Outliers) tespiti ve temizlenmesi.
   - Eksik verilerin (Missing Values) uygun yöntemlerle doldurulması.
   - Kategorik verilerin sayısal formata dönüştürülmesi (One-Hot / Label Encoding).
3. **Veri Görselleştirme:** Matplotlib ve Seaborn kütüphaneleri ile veriler arasındaki korelasyonlar ve dağılımlar görselleştirilmiştir.
4. **Modelleme ve Performans Karşılaştırması:**
   - 4 farklı Makine Öğrenmesi modeli eğitilmiştir.
   - **Cross-Validation (CV)** ve **Hyperparameter Tuning** işlemleriyle modeller optimize edilmiştir.
   
## Kullanılan Teknolojiler
- **Dil:** Python
- **Kütüphaneler:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
- **Yöntemler:** Data Cleaning, Feature Engineering, Model Tuning
